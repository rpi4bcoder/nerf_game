# nerf_game
A simple game based on NERF.

I wanted to make a game so I made this simple [SFML](https://www.sfml-dev.org/) window that includes a player and enemies.
The nerf idea came when I added `p_dir` to better show the player direction. I then made the colours blue and orangle and used the [Crillee Italic Std Regular](https://fontsgeek.com/fonts/crillee-italic-std-regular?ref=readme) from [fontsgeek.com](https://fontsgeek.com) for the cooldown text.

What you need:
g++ (coded in g++ 14)
[SFML](https://www.sfml-dev.org/) (coded in 2.6.2)
